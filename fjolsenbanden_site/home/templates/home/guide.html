Fjolsenbanden – Guide & How-To

Denne siden beskriver hvordan du kjører prosjektet lokalt for utvikling og hvordan du gjør en enkel produksjonsdistribusjon.

1. Forutsetninger

  • Python 3.11 eller nyere installert på maskinen din.
  • Et virtuelt miljø (venv) anbefales for lokal utvikling.
  • Opsjonelt for produksjon: Gunicorn som WSGI-server og en revers-proxy som Nginx.

2. Kjøring lokalt

  1. Klon repoet og gå inn i katalogen: 
    git clone <repo-url>
cd Fjolsenbanden
  2. Lag og aktiver et virtuelt miljø: 
    python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
  3. Installer avhengigheter fra requirements.txt: 
    pip install -r fjolsenbanden_site/requirements.txt
  4. Migrer databasen og start utviklingsserveren: 
    python fjolsenbanden_site/manage.py migrate
python fjolsenbanden_site/manage.py runserver
  5. Åpne nettleseren og gå til http://127.0.0.1:8000 for å se siden.

3. Kjøre tester

Prosjektet inkluderer en enkel test som bekrefter at forsiden rendres. Kjør testene slik: 

python fjolsenbanden_site/manage.py test

4. Produksjonsdistribusjon

For en lettvektsproduksjon kan du bruke Gunicorn bak en revers-proxy. Under ser du et eksempeloppsett på en Linux- server:

  1. Installer systempakker og aktiver miljøet ditt (for eksempel via systemd-tjenester).
  2. Kjør databasmigreringer før du starter applikasjonen: 
    python fjolsenbanden_site/manage.py migrate
  3. Start Gunicorn og pek på Django-prosjektets WSGI-applikasjon: 
    gunicorn --chdir fjolsenbanden_site fjolsenbanden_site.wsgi:application
  4. La en revers-proxy (for eksempel Nginx) motta trafikk på port 80/443 og videresende til Gunicorn på en intern port (for eksempel 127.0.0.1:8000).
  5. Konfigurer miljøvariabler for hemmelig nøkkel, tillatte verter og databindingspunkter via operativsystemet eller serverens systemd-tjeneste.

5. Nyttige kommandosammendrag

Oppdatere avhengigheter

Installer på nytt etter at requirements.txt endres.

pip install -r fjolsenbanden_site/requirements.txt

Samle statiske filer

Før produksjon, kjør collectstatic og gjør filene tilgjengelige via en webserver. 

python fjolsenbanden_site/manage.py collectstatic
